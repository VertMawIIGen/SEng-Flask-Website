What is this website?

The Software Engineering (SEng) Notes website is a comphrehensive website that contains everything that one possibly needs in order to ace the first HSC SEng task.

SEng Notes focuses on the syllabus topic Programming for the Web, thereby focusing on all key plot points that relate to website design, internet security, protocols, and other internet-related information.

This website also contains useful tools in order to practice the content learnt in an engaging and interactive manner.
